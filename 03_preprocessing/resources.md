# Preprocessing

Software and tools for preprocessing fMRI data prior to pRF model fitting.

## Data Organisation

| Tool | Description | Link |
|------|-------------|------|
| BIDS | Brain Imaging Data Structure standard | [bids.neuroimaging.io](https://bids.neuroimaging.io/) |
| HeuDiConv | Heuristic DICOM to BIDS converter | [github.com/nipy/heudiconv](https://github.com/nipy/heudiconv) |
| dcm2niix | Fast DICOM to NIfTI conversion | [github.com/rordenlab/dcm2niix](https://github.com/rordenlab/dcm2niix) |

## Preprocessing Pipelines

| Tool | Description | Link |
|------|-------------|------|
| fMRIPrep | Standardised fMRI preprocessing pipeline | [fmriprep.org](https://fmriprep.org) |
| FreeSurfer | Cortical surface reconstruction | [surfer.nmr.mgh.harvard.edu](https://surfer.nmr.mgh.harvard.edu/) |
| Neurodesk | Containerised neuroimaging platform | [neurodesk.org](https://www.neurodesk.org/) |
