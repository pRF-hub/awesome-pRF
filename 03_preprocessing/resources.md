# Preprocessing

Software and tools for preprocessing fMRI data prior to pRF model fitting.

## Data Organisation

| Tool | Description | Link | Tutorial |
|------|-------------|------|----------|
| BIDS | Brain Imaging Data Structure standard | [bids.neuroimaging.io](https://bids.neuroimaging.io/) | [BIDS conversion](https://neurodesk.org/edu/examples/workflows/bids_conversion.html), [PyBIDS](https://neurodesk.org/edu/examples/workflows/PyBIDS.html) |
| HeuDiConv | Heuristic DICOM to BIDS converter | [github.com/nipy/heudiconv](https://github.com/nipy/heudiconv) | [BIDS conversion](https://neurodesk.org/edu/examples/workflows/bids_conversion.html) |
| dcm2niix | Fast DICOM to NIfTI conversion | [github.com/rordenlab/dcm2niix](https://github.com/rordenlab/dcm2niix) | - |

## Preprocessing Pipelines

| Tool | Description | Link | Tutorial |
|------|-------------|------|----------|
| fMRIPrep | Standardised fMRI preprocessing pipeline | [fmriprep.org](https://fmriprep.org) | [fMRIPrep example](https://neurodesk.org/edu/examples/functional_imaging/fmriprep.html), [fMRIPrep on HPC](https://neurodesk.org/edu/tutorials/functional_imaging/fmriprep.html) |
| FreeSurfer | Cortical surface reconstruction | [surfer.nmr.mgh.harvard.edu](https://surfer.nmr.mgh.harvard.edu/) | [FreeSurfer example](https://neurodesk.org/edu/examples/structural_imaging/freesurfer.html), [FreeSurfer tutorial](https://neurodesk.org/edu/tutorials/structural_imaging/freesurfer.html), [mri_vol2surf](https://neurodesk.org/edu/examples/functional_imaging/mri_vol2surf.html) |
| Neurodesk | Containerised neuroimaging platform | [neurodesk.org](https://www.neurodesk.org/) | [Neurodesk demo tour](https://neurodesk.org/edu/examples/workflows/AA_Neurodesk_demo_tour.html) |
